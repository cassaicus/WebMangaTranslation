WebMangaTranslation is an macOS & iOS app that translates manga websites into your preferred language.  
This was created for technical verification.

![mwt](https://github.com/user-attachments/assets/7ac9eb9f-b921-447f-9ad0-12b5dd283498)


## Model

This application includes an object detection model trained using YOLOv8.
